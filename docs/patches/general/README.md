# Patches généraux

Ce dossier regroupe les patchs transverses, non limités à une session unique.

## Utilisation

- Placer les patchs `.diff` dans ce dossier.
- Appliquer les patchs depuis la racine du projet.
- Vérifier d’abord avec `git apply -p1 --check`.

## Exemples présents

- `PATCH_1_CORE_SECURITE.diff`
